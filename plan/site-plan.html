Overview

Purpose

The purpose of this site is to showcase my top-rated movies in a clean, visually appealing way that mirrors the functionality of Letterboxd. This site will highlight my personal taste in film, provide meaningful descriptions of why I like each movie, and allow visitors to explore my movie rankings.

Audience

The intended audience of this site is my classmates, my teacher, friends, and anyone who loves movies. The site is intended to reach those that love movies, but are also looking for movie recommendations from someone who has seen a lot of movies, me.

Dynamic elements

This website will feature a lot of dynamic elements using JavaScript to enhance interactivity:

    - A movie gallery generated from a JavaScript array of objects, where each object contains details such as title, year, genre, rating, and poster image.
    - A search bar and filter options that allow users to sort and filter movies by rating, release year, or genre using array methods like filter, map, and sort.
    - A Random Movie Recommendation feature that selects a film from the array and displays it using conditional logic.
